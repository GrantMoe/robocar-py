# RoboCar-Py

## Software
---
Python scripts to run on a Jetson Nano mounted on my 1/10 scale DIY Robocar(s).

Currently just a serial test. Cameras/GPS/autonomy to come later.

- [Fly Sky FS-GT5](https://www.flysky-cn.com/gt5-gaishu)
